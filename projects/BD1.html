Création d'une base de données

Introduction

L'objet de ce projet était d'étudier le naufrage du Titanic en se concentrant sur les victimes, et différents facteurs ayant une possible influence sur le taux de survie.

Pour ce projet nous étions donc en binôme.

Il a fallu dans un premier temps concevoir un Schéma Entité-Association, en se basant sur un set de données.

Nous avons ensuite créé les différentes tables, et effectué des séries de requètes pour obtenir différents éléments pour analyser la situation lors du nauffrage.
