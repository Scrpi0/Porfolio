Installation d'un poste de développement

L'objectif fut d'installer un poste adapté pour le développement , en l'occurence du développement avec le language en java.

Il s'est deroulé en plusieurs phases:

  • installation d'un système d'exploitation (GNU/Linux distributon Debian)
  • installation de magasins d'applications (snap & flatpack)
  • installation d'un IDE (intellij idea)
  • installation d'un Java Developement Kit

À l'issue de cette installation, les différentes informations nécessaires à cette dernière ont été résumées à l'aide d'une Carte Mentale.
