Accidentologie

Introduction

L'accidentologie est un domaine qui dans notre cas a necessité de lier différentes compétences.

D'une part en SQL, il nous était fournis plusieurs jeux de données concernant les accidents en France durant ces dernières années.

Avec mon binôme nous avons dû formatter les données pour qu'elles puissent être traitées indifféremment, puis nous avons dû extraire les données que nous recherchions, celles concernant les personnes de plus de 65 ans en nouvelle Aquitaine.

Puis nous nous sommes ensuite servis de ces données avec R pour réaliser différents graphiques sur ces accidents, dans notre dommaine d'étude.
